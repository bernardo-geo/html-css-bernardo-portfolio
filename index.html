    Development of Reserva Ecológica Nacional (REN)

    Involved in developing RENs for the municipalities of Guarda and Sertã. The REN has helped to protect natural resources, safeguard processes that are essential for good land management and promote the conservation of nature and biodiversity, essential components of our country's biophysical support (CCDRC, I.P.).

    Developed using Arcgis Pro and QGIS software

    Definition of Universal Soil Loss Equation (USLE) and multi-criteria decision analysis (AHP)

    The susceptibility to water erosion of the soil in the Leiria district was assessed using the Universal Soil Loss Equation (USLE) and AHP multi-criteria analysis. The geoprocessing was planned, organized and executed using the Model Builder manager in ESRI's ArcGIS Pro software.

    Analysis of the potential consequences of an earthquake in the Lisbon region

    SThe current impact of the 1755 tsunami on the population and buildings in Lisbon parishes was simulated. An evacuation plan was proposed for the main affected squares and vulnerability and hazard maps were obtained.

    Developed using Arcgis Pro software

    Land use and occupation map - remote sensing project - Part I

    Changes in land use and occupation in Itabira, Brazil, were analyzed using Sentinel-2 images from 2018 and 2023. Satellite image processing steps were applied, including pre-processing, band transformation and change detection via NDVI. Different spectral signatures were recognized, and consequently different nomenclature classes were defined. The processing made it possible to draw up land use and land cover maps.

    Developed using Arcgis Pro software

    NDVI arithmetic difference analysis map - remote sensing project - Part II

    MNDVI arithmetic difference analysis map, where the three values represent an increase (1), no change (0) and a decrease (-1) in NDVI. Detail of the temporal difference in the mine area. Changes in land cover - decline in vegetation (negative values), due to the expansion of the exploration area, thus allowing the variability between the different years to be assessed.

    Developed using Arcgis Pro software

    Comparison of interpolation methods: IDW, OK e EBK

    Spatial interpolation methods Inverse Distance Weighting (IDW), Ordinary Kriging (OK) and Empirical Bayesian Kriging (EBK) for precipitation data from 105 weather stations. Local Moran's I and Global Moran's statistical tools were used, as well as Arcgis Pro's Exploratory Interpolation tool to assess the best interpolation method, and the Geostatistical Wizard was used to improve these results. Arcgis Pro's Focal Statistics and Block Statistics tools were used to analyze the IDW.
